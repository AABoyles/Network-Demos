# Basic SVG D3 WASM Force-directed Graph

A D3 WASM Force Directed Graph with Zooming and Dragging. Based on the [d3-wasm-force](https://github.com/ColinEberhardt/d3-wasm-force) library by Colin Eberhardt. See also [the associated Blog post on ScottLogic](https://blog.scottlogic.com/2017/10/30/migrating-d3-force-layout-to-webassembly.html)
